# Number Toolkit

## 📌 Description

This project is a simple Python program that analyzes a list of numbers.
It demonstrates basic programming concepts such as functions, loops, and conditionals.

The program calculates:

* Sum of numbers
* Maximum value
* Minimum value
* Count of even numbers

---

## 🧠 Features

* Uses functions (`def`)
* Uses `while` loops
* Works with lists
* Applies conditional logic (`if`)

---

## 📂 Project structure

```
number-toolkit/
 ├── main.py
 └── README.md
```

---

## 🚀 How to run

1. Make sure Python is installed
2. Run the program:

```
python main.py
```
